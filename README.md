### Ⅰ. **核心架构**

#### ◈ 时空基底

- **主世界·蓝星**：正经历「灵枢归位」大劫，地脉中爬出上古谶言："九棺镇世，今缺其三"
- **暗域大世界**：算法师与封灵使者的主场，星界精铁与云鲸泪晶构成能量网络
- **诸天裂隙**：每49天开启随机文明残影副本（埃及/玛雅/赛博等）

#### ◈ 终极宿命

- 主角为上一世界线斩杀星神雅特的唯一幸存者，背负《时茧哀歌》逆转时空归来
- 右手背星轨刻痕每日缩减1cm（归零时雅特残影吞噬现世）
